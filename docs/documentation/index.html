Introduction

This page serves as a reference to the public API for the Game Texture Loader and the types used there in.

The API is pretty simple, however when using the C++ interface its best to pay attension the exceptions which can be issued from a couple of the functions.

New users to the library will want to read about the Key Concepts, which explain how to configure the GTL when recompiling and about the boost style auto-linking which is avalible.
